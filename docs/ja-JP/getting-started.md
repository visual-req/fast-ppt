# はじめに

![Outline Detail Workflow](../assets/ja-JP/outline-detail-flow.svg)

## クイックスタート

### 1. 入力の準備

プロジェクト資料を次の場所に配置：

```text
work/input/001_プロジェクト名/
```

入力は要件説明、原稿、コース概要、提案書など、任意の形式で構いません。

### 2. アウトライン生成

Skill の第 1 フェーズを実行：

```text
/fppt:outline
```

このフェーズで以下が生成されます：

```text
work/ppt/001_プロジェクト名/outline.json
```

### 3. 構造の確認

詳細ページを生成する前に、以下を確認：

- 章が完全か
- ページ順序が適切か
- ページの意図が明確か
- 図版ページが十分に計画されているか

### 4. 詳細ページの生成

確認後、以下を実行：

```text
/fppt:detail
```

このフェーズで以下が生成されます：

- `deck.json`
- `slides/*.json`
- `work/assets/*.svg`

### 5. プレビューを開く

ローカルサーバーを起動し、ブラウザで開く：

```text
http://localhost:9030/
```

最新ビルドが反映されない場合は、ブラウザを強制リフレッシュしてください。
