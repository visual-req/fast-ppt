# トラブルシューティング

## ページ表示がおかしい場合

### 症状 1：レイアウトが間違っている

例：比較ページがステップ図で表示される、ワークフローがテーブルで表示される、階層構造が bullets のまま

**修正方法**：
1. 該当ページの `slides/*.json` を開く
2. `layout_type` を確認
3. 正しいレイアウトに変更

一般的な対応：
- 比較 → `comparison_table`
- フェーズ → `phases`
- ワークフロー → `swimlane_process`
- 構造図/関係図/階層図 → `svg_full`

### 症状 2：タイトルが間違っている/表示されない

**修正方法**：
1. ページ JSON の `title` を確認
2. `svg_full` ページの場合、`show_title: true` が設定されているか確認
3. ページタイトルを SVG 内に重複させない

### 症状 3：SVG が正しく表示されない

**修正方法**：
1. `work/assets/*.svg` を開く
2. SVG のテキスト・構造・寸法を直接編集
3. ページ JSON の `svg.src` が正しいファイルを指しているか確認
4. ブラウザを強制リフレッシュ

### 症状 4：コンテンツが画面からはみ出る

**修正方法**：
1. bullets 密度を下げる
2. 本質的に構造図なら `svg_full` に変更
3. フェーズチェーンなら `phases` に
4. プロセスなら `swimlane_process` に

## 推奨調査順序

1. ページ JSON が正しいか確認
2. 参照 SVG が正しいか確認
3. フロントエンドのレイアウトコンポーネントを確認
4. 最後にプロンプトを修正して再発防止
